# Merge Discontinuous Time Ranges

This Node.js module merges overlapping or nearly continuous time ranges based on a given threshold.

---

## How to Run

1. **Install Node.js** (v14+ recommended)

2. **Clone or download this repository**

   ```bash
   git clone https://github.com/ankushchhabra02/saralweb.git
   cd merge-time-ranges
   ```

3. **Run the test file**
   ```bash
   node test.js
   ```
